# In da Club

## Description

NeuroSync Corp's "Executive Lounge" is an exclusive digital space for board members and high-ranking officials. It is protected by a "state-of-the-art" access control system that allegedly verifies user roles to grant access.

Rumor has it that the developers are working on a better access system after netrunners breached the previous one. However, the old vulnerability might still be active.
